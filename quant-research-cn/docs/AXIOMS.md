# Mathematical Axiom System — A-Share Adaptation

The entire screening system derives from **5 axioms**, identical in structure to the US pipeline.
The only difference: which observable signals instantiate each axiom.

## Axiom 1 — Conditional Probability

> P(r > 0 | state) ≠ P(r > 0)

Regime classification makes unconditional return probabilities useless.

**US instantiation**: regime (autocorrelation) × vol_bucket → 9-cell CPT
**A-share instantiation**: identical. Price statistics are market-agnostic.

```
regime ∈ {trending, mean_reverting, noisy}     ← lag-1 autocorr over 20 bars
vol_bucket ∈ {low, mid, high}                  ← intraday vol / 20d avg, terciles
cell(regime, vol_bucket) → Beta-Binomial posterior → P(5D return > 0 | cell)
```

## Axiom 2 — Bayesian Updating

> Prior × Likelihood → Posterior

Beta(α₀, β₀) + observed (wins, losses) → Beta(α₀+w, β₀+l)

- **Prior**: Beta(2, 2) — mildly informative, avoids extreme posteriors on small samples
- **Posterior mean**: (α₀ + w) / (α₀ + β₀ + w + l)
- **95% credible interval**: Beta quantiles [0.025, 0.975]

Applied to:
| Module | wins / losses definition |
|--------|------------------------|
| `momentum` | 5D return > 0 in same CPT cell |
| `announcement_risk` | 5D excess return > 0 after 业绩预告 by type |
| `flow_score` | 5D return > 0 when northbound net buy > 1σ |
| `unlock_risk` | 5D return < 0 in [T-5, T+5] around unlock date |

## Axiom 3 — Latent States (HMM)

> Observable returns are generated by hidden market regimes

2-state Gaussian HMM on benchmark (沪深300) daily returns:
- **State 0 (bull)**: μ > 0, lower σ
- **State 1 (bear)**: μ < 0, higher σ

Outputs:
- `P(bull)` — current regime probability
- `P(r > 0 | tomorrow)` — forward prediction via transition matrix
- `regime_duration` — days in current regime
- `brier_score` — calibration metric (recorded daily, resolved next day)

**Calibration lifecycle**: same as US.
```
Day T:   fit HMM → predict P(r>0 | T+1) → INSERT forecast
Day T+1: observe actual return → UPDATE forecast resolved=true
Weekly:  compute Brier score over last 60 resolved forecasts
```

## Axiom 4 — Multi-Source Information Fusion

> No single signal is sufficient. Combine orthogonal information.

**US pipeline** (5 dimensions):
```
composite = w₁·magnitude + w₂·event + w₃·momentum + w₄·options + w₅·cross_asset
```

**A-share pipeline** (5 dimensions, adapted):
```
composite = w₁·magnitude + w₂·event + w₃·momentum + w₄·flow + w₅·cross_asset
```

| Dimension | US signal | A-share signal |
|-----------|-----------|---------------|
| magnitude | |5D return| z-score | identical |
| event | earnings surprise + 8-K filings | **业绩预告类型 + 公告** |
| momentum | CPT trend_prob | identical |
| options/flow | IV ratio + P/C + skew | **北向净买入 z-score + 融资余额变化 + 大宗折溢价** |
| cross_asset | sector ETF + VIX + bonds | **行业ETF + 上证50/创业板比值 + 国债** |

### Flow Score (replaces Options Score)

```
flow_score = 0.30 × northbound_zscore      # 北向资金净买入标准化
           + 0.25 × margin_delta_zscore     # 融资余额5日变化标准化
           + 0.20 × block_trade_premium     # 大宗交易溢价率
           + 0.15 × hot_money_intensity     # 龙虎榜游资净买入/流通市值
           + 0.10 × turnover_ratio_zscore   # 换手率异常
```

### Announcement Risk (replaces Earnings Risk)

业绩预告 types map to surprise categories:
```
预增/扭亏     → positive surprise → P(5D excess > 0 | type) via Beta-Binomial
预减/首亏     → negative surprise
续盈/略增     → mild positive
续亏/略减     → mild negative
不确定        → neutral
```

## Axiom 5 — Finite Attention

> Human attention is bounded. Saturate, then filter.

Same two-pass architecture:
```
Pass 1: full universe (~300-1000) → rank by composite → top 120 candidates
Pass 2: candidates + flow data + announcements → re-rank → top 30 notable items
```

Classification: HIGH / MODERATE / WATCH / LOW (identical to US)

Saturation clamp: `score = min(score, 1.0)` — prevents any single signal from dominating.

## Macro Gate Matrix (A-share Adapted)

3×3 matrix: **上证波动率** × **期限利差 (10Y-1Y国债)**

| | Calm (< 20%) | Elevated (20-35%) | Panic (≥ 35%) |
|---|---|---|---|
| **Normal spread** | 1.0× | 0.9× | 0.7× |
| **Flat/Inverted** | 0.9× | 0.8× | 0.6× |
| **Steep** | 1.1× | 1.0× | 0.8× |

Per-asset-class multipliers:
- 科技/成长: ×1.2 in calm, ×0.7 in panic
- 消费/红利: ×0.9 in calm, ×1.1 in panic
- 周期/资源: follows commodity overlay

## Unique A-Share Modules (no US equivalent)

### Unlock Risk (限售解禁)
```
P(5D return < -2% | unlock_ratio, days_to_unlock) via Beta-Binomial

unlock_ratio = shares_unlocking / float_shares
buckets: small (<1%), medium (1-5%), large (>5%)
window: [T-5, T+5] around unlock date
```

### Hot Money Score (龙虎榜)
```
net_buy_ratio = (buy_amount - sell_amount) / circulating_market_cap
institutional_ratio = institutional_buy / total_buy
→ signal when net_buy_ratio > 2σ AND institutional_ratio > 30%
```

### Northbound Flow Regime
```
Rolling 20D northbound net flow → 3-state HMM:
  State 0: accumulation (外资加仓)
  State 1: distribution (外资减仓)
  State 2: neutral
→ P(accumulation) as overlay multiplier
```
